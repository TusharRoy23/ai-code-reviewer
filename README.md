# AI Code Reviewer Bot

Get **expert-level code reviews** on every PR — powered by GPT-4o, Claude 3.5, or Grok.

Detects bugs, security flaws, performance issues, style problems, and more — all in seconds.

### Install in 30 seconds

Click this button → create a new repo from template → done!

[![Use this template](https://img.shields.io/badge/Use%20this%20template-2ea44f?logo=github&style=for-the-badge)](https://github.com/your-username/ai-code-reviewer/generate)

### Setup

After creating your repo from template:

1. Go to **Settings → Secrets and variables → Actions**
2. Add one of these secrets:
   - `OPENAI_API_KEY` → for GPT-4o (recommended)

That's it!
